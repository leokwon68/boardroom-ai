# Boardroom — 완벽 가이드

**1인 회사의 법인화.** 당신을 학습한 AI 이사회가 스스로 안건을 찾아 회의하고, 근거로 싸워 확신도와 함께 판결하고, 자기 판단을 현실과 대조해 채점합니다 — 그리고 executor가 버튼 하나에 진짜로 실행합니다. 브라우저로도, 폰으로도 굴립니다.

---

## 1. 설치 & 실행

**Node 18+** 만 있으면 됩니다.

```bash
npx boardroom-ai              # 앱 실행 + 브라우저 자동 오픈 → http://localhost:4242
npx boardroom-ai "가격 20% 올릴까?"   # 또는 터미널에서 회의 1번
```

소스에서:

```bash
git clone <repo> && cd boardroom && node server.mjs
```

끄려면 터미널에서 `Ctrl+C`.

---

## 2. 사람들이 헷갈리는 2가지 (축이 2개다)

### 어디서 도느냐
- **로컬 (지금):** 내 컴퓨터에서 도는 프로그램. 브라우저로 `localhost:4242` 접속. 나 혼자 씀. 로그인 없음. 무료.
- **호스팅 (미래):** 로그인하는 웹사이트. 우리가 클라우드에서 24/7 돌려줌. 소프트웨어가 아니라 *편의*에 과금.

### AI 두뇌를 뭘로 굴리느냐
- **클로드 구독** — `claude` CLI가 깔려 로그인돼 있으면 그걸로 굴러감. **토큰 추가비 0원.**
- **API 키** — Anthropic / OpenAI / Gemini. 쓴 만큼 그 회사에 냄. CLI가 없을 때만 사용.

> **AI 비용은 모델 회사에 내는 것이지 우리에게 내는 게 아닙니다.** 클로드 구독 있으면 사실상 공짜. (미래) 호스팅 요금은 "우리가 24/7 대신 켜둠"의 값일 뿐.

키는 이 컴퓨터에서 **자동 감지**됩니다. 아무것도 없을 때만 한 번 붙여넣으면 됨 — 로컬(권한 `600`) 저장, 다시 표시 안 됨.

## 3. 탭별 사용법

**⚙ Engine** — 키/구독 자동 감지 확인. 역할별 모델 배정: 의장 Fable, 좌석 Sonnet, GPT 좌석 혼합 — 자유.

**👥 Staff** — 이사 채용. 각 이사는 모든 결정을 렌즈 하나로 읽음(법무·그로스·CFO). 이사회는 구성만큼 날카로움. *회사가 나에 대해 아는 것*(학습된 프로필)도 여기서 봅니다.

**▦ Boardroom** — 진짜 결정 하나 입력 → **Convene**. 이사들이 순서대로 주장·반박하고, 의장이 확신도와 함께 판결하고, 레드팀이 그 판결을 죽이려 듭니다. 약 90초.
- **⚡ Autopilot** (드롭다운 하나: 끔 / 30분 / 매시간 / 3시간): 이사회가 **스스로** 당신 관심사로 회의 — 클릭 불필요. 실행 계획을 Approvals에 쌓음. **Autopilot Activity** 피드에서 실시간 관전.

**✓ Approvals** — 이사회가 하려는 모든 일이 여기 대기. 버튼 둘: **Execute / Decline**. Execute → executor가 진짜 브라우저(Playwright)·셸·파일로 수행하며 각 단계를 말로 중계. 위험한 단계(결제·발송·로그인·매매)는 영원히 사람 승인 너머.

**📈 Ledger** — 모든 판결이 리뷰일과 함께 기록 → 날 되면 ✓적중 / ✗빗나감 채점. 이사회의 **타율**이 쌓임. 자기 판단을 측정하는 유일한 AI.

**📄 Reports** — 모든 회의록 보관. 자리 비운 동안에도 autopilot이 계속 추가.

사이드바 하단: **🔊 사운드** 토글(이사 발언·판결 시 미세 효과음) + **EN / 한국어** 언어.

## 4. 폰에서 굴리기 (텔레그램)

Boardroom이 폰으로 알려주고, 폰에서 명령도 받습니다 — 비서실장처럼.

**설정:** `~/.boardroom/config.json` 에 `telegram` 블록 추가:

```json
{
  "telegram": { "botToken": "123456:ABC...", "chatId": "-1001234567890", "threadId": 4 }
}
```

봇 토큰은 **@BotFather** (`/newbot`)에서 발급, 그 봇을 채팅/그룹에 추가하고 `chatId`를 넣으세요. `threadId`는 선택(포럼형 그룹만).

**폰으로 오는 것:**
- 🏛 이사회가 스스로 회의하면 → 판결 + 승인 대기 계획 id

**봇에게 보낼 수 있는 것:**
- `help` — 명령 목록
- `status` — 승인 대기 목록
- `go <id>` — 큐에 쌓인 계획 실행 (진짜 executor 가동)
- `no <id>` — 계획 거절
- *그 외 아무 텍스트* — 그 결정으로 이사회 소집 → 판결 회신

→ 어디서든 진짜 작업을 승인하거나, 결정을 요청할 수 있습니다.

## 5. 전체 루프

```
회사가 나를 학습 → 관심사로 계열사 설립 →
스스로 회의 소집 → 토론 → 판결(확신도 + 반증조건) →
실행 계획 → [ 나: Execute / Decline ] → 진짜 실행(브라우저·셸·파일) →
결과 정리 → 판결을 현실과 대조 채점 → 타율
```

당신은 오퍼레이터에서 **승인자**로 바뀝니다.

## 6. FAQ

**가입할 사이트가 따로 있나?** 아직 없음. 지금은 로컬 프로그램 — `npx boardroom-ai`. 호스팅판(로그인·상시가동)은 나중에.

**로컬은 왜 로그인이 없나?** 내 컴퓨터에서 나만 쓰니까. 로그인은 호스팅 멀티유저판에만 필요.

**돈 드나?** 소프트웨어는 무료. AI 두뇌는 모델 회사가 과금(클로드 구독이면 공짜). 우리는 미래 호스팅 편의에만 과금.

**시키지 않아도 뭘 하나?** 스스로 *판단·계획*은 하지만, 내 Execute 없이는 절대 실행 안 함. 결제·발송·로그인·매매는 무조건 사람 몫.

**내 데이터는?** 전부 로컬 `~/.boardroom/` (회의록·원장·오너프로필·큐). 내가 설정한 모델 API 호출 말고는 기계를 떠나지 않음.
